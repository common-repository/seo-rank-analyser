SEO Rank Analyser (Analyzer) is free and light weight one stop solutions for all SEO (Search Engine Optimization) related analysis and advice. With this  plugin, you can analyze on-page SEO, website score and All page (post and pages) rank checker. This is best SEO Audit tool.

== Description ==
SEO Rank Analyser (Analyzer) is one stop powerful solution for On-page SEO test. This plugin check and suggest you for every elements of your website with description.  You can analyze your home page and inner pages in one click. 

You can contact us at  support @ seorankanalyser.com  for support.

<h4>How it works</h4>

This plugin just send your domain (URL) to http://seorankanalyser.com/ web site and get details data from server. All data processed in http://seorankanalyser.com and return results to your plugin's panel.
